Uploading Everything

The next thing you need to do is upload all files to your web server. To do this, first connect to your web server via FTP. If you do not know how to use File Transfer Protocol (FTP), please click here to go to the FTP tutorial we included with this manual. Once logged in to your web server, please follow these steps.

  1. Upload the ezylist_data directory

    1. Create a directory on your server called ezylist_data. For security reasons, we recommend you put this directory below your document root, which is where you should be right now.

    2. Upload the contents of the ezylist_data directory, from the zip file, in to the ezylist_data directory you just created. Make sure you upload everything in ASCII mode!

    3. CHMOD the ezylist_data directory as follows:

      • ezylist_data - 777
      • ezylist_data/dat - 777
      • ezylist_data/htmlt - 777
      • ezylist_data/ezylist.conf - 777
      • ezylist_data/ezylist.cron - 777
      • ezylist_data/dat/ (All .dat files inside directory) - 777
      • ezylist_data/htmlt/admin - 755
      • ezylist_data/htmlt/cpanel - 755


  2. Upload the cgi-bin directory

    1. Change directories on your web server to your cgi-bin directory, where all of your CGI scripts are stored.

    2. Create a new directory inside your cgi-bin called ezylist

    3. Upload the contents of the cgi-bin directory, from the zip file, in to the cgi-bin/ezylist directory you just created. Make sure you upload everything in ASCII format!

    4. CHMOD the cgi-bin directory as follows:

      • cgi-bin/error.cgi - 777
      • cgi-bin/followup.pl - 777
      • cgi-bin/install.cgi - 755
      • cgi-bin/newuser.cgi - 777
      • cgi-bin/process.pl - 777
      • cgi-bin/reminder.cgi - 777
      • cgi-bin/subscribe.cgi - 777
      • cgi-bin/update.pl - 777
      • cgi-bin/admin - 755
      • cgi-bin/admin/.htaccess - 777
      • cgi-bin/admin/admin.cgi - 777
      • cgi-bin/admin/ (All .pl files inside directory) - 755
      • cgi-bin/cpanel - 755
      • cgi-bin/cpanel/.htaccess - 777
      • cgi-bin/cpanel/ (All .pl files inside directory) - 755
      • cgi-bin/lib - 755
      • cgi-bin/lib/ (All .pl files inside directory) - 755
      • cgi-bin/modules - 755
      • cgi-bin/modules/exec.pl - 777

      NOTE: If you are familiar with installing CGI scripts, you are probably thinking that some of the permissions above are wrong. They are NOT! Please CHMOD all of the files to the exact permissions as stated above, or the script will not install correctly.


  3. Upload the images directory

    1. Change directories on your server to your images directory, where all of your images are stored. If you do not currently have one, create a directory called images inside your document root.

    2. Inside your images directory, create a new directory called ezylist

    3. Upload the contents of the images direcrory, from the zip file, in to the new images/ezylist directory you just created. Make sure you upload everything in Binary format, NOT ASCII!

    4. You do NOT need to CHMOD any of the files in the images directory.
