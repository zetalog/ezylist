Telnet Tutorial - The which and whereis commands.

Almost all CGI scripts which you install, require that you know the full path to various programs on your web server such as perl and sendmail. This is what the which and whereis commands are for. The basic syntax is:
which NAME
which perl
whereis perl

Basically, you just type which or whereis then the program name, like perl or sendmail. You should ALWAYS use the which command instead of whereis because the which command will give you the servers default location, and whereis will give you all locations of that program. Only use the whereis command if the which command doesn't display anything.

Remember, these two commands can be used for any program on your server. So you could have things like which perl, which sendmail and which procmail.
