Telnet Tutorial - Deleting Files and Directories

By now, you should be able to easily move around your web server, find the full path on your server, and CHMOD files and directories. Next, we will learn how to delete entire files and directories with one simple UNIX command. The basic syntax is:


rm -rf FILENAME

This command will delete the file name you specify. If you enter the name of a directory, the directory and everything inside of it will be automatically deleted.
