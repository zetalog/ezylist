Telnet Tutorial - Unzipping tar.gz files

Have you ever downloaded a zip file which has a tar.gz extension? This is a UNIX zip file. To unzip this file, upload it to your web server, then telnet into your account. First you have to unzip the file which you can do by typing:
gunzip FILENAME
gunzip file_tar

Do NOT type the ending .gz when your unzipping the file. The server already knows it's supposed to be there. This will unzip the FILENAME, and leave you with another, much larger, file without the .gz extension. To unzip this file, you will have to type the following:
tar -xvf FILENAME
tar -xvf file_tar

If your wondering what -xvf means, the x means extract a file, the v means verbose so you can see what's happening, and f means the filename. This will unzip the entire contents of the FILENAME and create any directories needed.
