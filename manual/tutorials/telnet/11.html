Telnet Tutorial - Conclusion

This concludes the telnet tutorial. We hope you've learned a lot about connecting to your web server using telnet. You should now know how to do the following:

  • Connect to your server through telnet
  • Navigate around your web server
  • Find the full path to any directory on your web server
  • Change the permissions of files and directories (CHMOD)
  • Delete, copy and move files and directories
  • Run and debug CGI scripts
  • View the contents of a file
  • Edit a file using the pico editor
  • Find the full path to various programs such as perl and sendmail
  • Unzip tar.gz files

Thanks a lot for taking the time to read and learn out tutorial!
