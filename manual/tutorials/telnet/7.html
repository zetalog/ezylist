Telnet Tutorial - Viewing the Contents of a File

In telnet, there is a very easy way to view the contents of a file on your web server. This is called the cat command. To use the cat command, at the telnet prompt type:
cat FILENAME
cat script.cgi

This will display, on the telnet screen, the contents of FILENAME. Unfortunately, some files are a little too large for this, or what happens if you only want to see the first 30 lines of the file? There's another UNIX command called head which will only print out the specified number of lines starting at the beginning of the file. The basic syntax for the head command is:
head -n FILENAME
head -30 script.cgi

This will display the first n lines of the file FILENAME. If -n is left out, only the first 20 lines will be displayed. Now what happens if you want to view only the bottom 20 lines of a file? Once again, there's a UNIX command which will do this for you. It's called the tail command. The basic syntax of the tail command is:
tail -n FILENAME
tail -20 script.cgi

This will display the last n lines of the file FILENAME. If -n is omitted, only the last 15 lines will be displayed.
