Telnet Tutorial - Copying and Moving Files and Directories

Now you know how to delete files and directories, so lets learn how to copy and move them. Copying and moving files in telnet is just as easy as deleting them. The basic syntax for copying a file or directory is:
cp -R OLDFILE NEWFILE

cp -R /home/mysite/www/members /home/mysite/www/reps

This command will copy whatever you put for OLDFILE to whatever you put for NEWFILE. In the example above, everything inside the directory /home/mysite/www/members would be copied over to the directory /home/mysite/www/reps. Both directories will still exist, and will contain the exact same thing. This command will work for both, files and directories.

Moving files and directories is done in almost the exact same way. The basic syntax to move a file or directory is:
mv OLDFILE NEWFILE

mv /home/mysite/www/members /home/mysite/www/reps

The above example will move the directory /home/mysite/www/members to the directory /home/mysite/www/reps. The members directory will no longer exist, unlike copying the directory. This command will also work on both, files and directories.

To rename a file or directory, you would also use the mv command. Since this command moves the OLDFILE to the NEWFILE, it is just like renaming a file or directory.
