Telnet Tutorial - The CHMOD command

Another thing which you are required to do when installing any CGI script, is to CHMOD or change the permissions of a file or directory. Usually, and CGI scripts with a .cgi or .pl ending are chmoded to 755, and most other text files should be chmoded to 777.

To CHMOD files and directories in telnet is very easy. At the telnet prompt, all you have to do is type:
chmod PERMISSIONS FILENAME

chmod 755 script.cgi

In the above example, the file called script.cgi would have been chmoded to 755.
